댄스 영상 업로드 STEP 1
분석할 원본 댄스 영상(mp4·mov·webm) 업로드. 영상은 브라우저에서 직접 처리 — Claude가 균등 간격으로 프레임 자동 추출
⚡ 30초~2분, 1080p 이하 권장. 영상 자체는 외부로 전송되지 않음
↓
3
AI 댄스 분석 STEP 2 → 분석 시작
Claude Vision이 프레임을 보고 안무를 구간(Section)으로 분류 → 각 구간의 에너지·흐름·전환 방식 파악 → 구간 내 세부 동작 추출 (이름·자세·방향·연결법)
⚡ 동작 수 선택: 16개(기본) / 20개 / 30개. 장르 선택이 분석 정확도에 영향
↓
4
GPT Image 2 — 댄스 시트 생성 ChatGPT
GPT Image 2 프롬프트 복사 → ChatGPT 열기 → 프롬프트 붙여넣기 → 이미지 생성 → 저장. 시트에는 구간 헤더 + 동작 번호 + 자세 그림 + 방향 화살표가 포함됨
⚡ 캐릭터 레퍼런스 이미지를 ChatGPT에 함께 첨부하면 일관성 ↑

⚡ @ 버튼으로 캐릭터 Omni·배경 레퍼런스 등을 프롬프트에 직접 삽입 가능

Seedance 2.0 — 영상 생성 higgsfield.ai
Seedance 열기 → img1 = GPT Image 2로 만든 댄스 시트 / img2 = 캐릭터 시트 / aud1 = 음악 파일 첨부 → 프롬프트 붙여넣기 → 생성
⚡ 프롬프트 권장 길이 60~100 단어. 카메라 지시는 1개만. 기술 스펙(fps·ISO) 사용 금지

⚙️ 동작 원리 — 영상은 브라우저에서 직접 재생 처리됩니다. Claude API는 영상을 직접 받지 않으며,
대신 영상에서 프레임(스틸 이미지)을 자동 추출해 분석합니다.
추출된 각 프레임은 JPEG로 압축되어 API에 전송 (프레임 1장당 5MB 이하, 최대 8장 전송).
⚠️ 실질적 제한 — 영상 자체 용량보다 브라우저 메모리(RAM)가 병목입니다.
4K·60fps 고용량 영상은 브라우저에서 프레임 추출 시 느릴 수 있으므로 1080p 이하 권장.
